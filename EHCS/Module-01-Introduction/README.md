# Module 01 - Introduction to Ethical Hacking

## Overview

This module introduces the fundamentals of Information Security, Cyber Security, Ethical Hacking, the CIA Triad, Information Warfare, and the role of Ethical Hackers. It serves as the foundation for all upcoming cybersecurity modules.

---

## Topics Covered

- Information & Information Security
- Information Security Threats
- Cyber Security
- Objectives of Cyber Security
- CIA Triad
- Hacking Fundamentals
- Types of Hackers
- Ethical Hacking
- Information Warfare
- Need for Ethical Hackers
- Career Opportunities in Cyber Security

---

## Learning Outcomes

After completing this module, I can:

- Differentiate Information Security and Cyber Security.
- Explain the CIA Triad.
- Understand Threats, Vulnerabilities and Risks.
- Identify different types of Hackers.
- Explain Ethical Hacking and its importance.
- Understand Information Warfare.
- Describe the role of Ethical Hackers.

---

## Key Takeaways

- Information is a valuable asset.
- Cyber Security protects digital assets.
- CIA Triad is the foundation of Information Security.
- Ethical Hackers legally discover security weaknesses.
- Cyber Security is a rapidly growing career field.
